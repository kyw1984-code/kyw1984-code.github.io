개인정보처리방침

방과후시간표 (Afterschool Schedule)

시행일: 2026년 5월 3일  |  최종 개정: 2026년 5월 3일

1. 개인정보 수집 항목 및 목적

방과후시간표(이하 '서비스')는 다음과 같은 개인정보를 수집·이용합니다.

수집 항목수집 목적보유 기간
이메일 주소, 비밀번호(암호화) 회원 가입 및 로그인 인증 회원 탈퇴 시까지
닉네임(선택) 서비스 내 사용자 식별 회원 탈퇴 시까지
자녀 이름, 색상 태그 자녀별 일정 구분·관리 회원 탈퇴 시까지
방과후 활동 일정(활동명, 요일, 시간, 장소) 시간표 생성 및 가족 공유 회원 탈퇴 시까지
사교육비 정보(금액, 결제일, 카테고리) 월별 비용 집계 및 분석 회원 탈퇴 시까지
기기 식별 토큰(FCM Token) 푸시 알림 발송 알림 해제 또는 회원 탈퇴 시까지

서비스는 위 목적 이외의 용도로 개인정보를 이용하지 않습니다. 목적이 변경될 경우 사전에 동의를 받겠습니다.

2. 개인정보 수집 방법

  • 이메일/비밀번호 회원 가입 시 직접 입력
  • 앱 사용 중 자녀 정보·일정·비용 데이터 직접 입력
  • 기기로부터 자동 수집 (FCM 기기 토큰)

3. 개인정보의 제3자 제공

서비스는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 서비스 운영을 위해 아래 위탁 처리를 합니다.

수탁자위탁 업무보유 기간
Google LLC (Firebase) 사용자 인증, 데이터베이스 저장, 파일 스토리지, 푸시 알림(FCM) 위탁 계약 종료 시까지

Firebase의 개인정보 처리방침: https://policies.google.com/privacy

4. 개인정보 보유·이용 기간 및 파기

수집된 개인정보는 이용 목적이 달성되거나 회원 탈퇴 요청 시 즉시 파기됩니다.

  • 전자적 파일: 복구 불가능한 방식으로 영구 삭제
  • 기타 기록물: 분쇄 또는 소각

단, 관련 법령에 의해 보존이 필요한 경우 해당 기간 동안 보관 후 파기합니다.

5. 이용자의 권리 및 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 오류 정정 요청
  • 개인정보 삭제 요청 (회원 탈퇴)
  • 개인정보 처리 정지 요청

앱 내 설정 → 회원 탈퇴 기능을 통해 즉시 처리하거나, 아래 이메일로 요청하실 수 있습니다.

6. 개인정보 보호를 위한 기술적·관리적 조치

  • 비밀번호는 Firebase Authentication이 암호화하여 저장 (평문 저장 없음)
  • 모든 데이터 전송은 HTTPS(TLS) 암호화 적용
  • Firebase Security Rules를 통해 인가된 사용자만 자신의 데이터에 접근 가능
  • 가족 구성원 간 데이터 공유는 초대 코드 기반 명시적 동의 후에만 허용

7. 아동 개인정보 보호

서비스는 만 14세 미만 아동이 직접 가입하여 이용하는 서비스가 아닙니다. 앱에 입력되는 자녀 정보(이름, 활동 일정 등)는 보호자가 직접 입력·관리하며, 자녀의 개인정보는 보호자의 관리 하에 처리됩니다.

8. 개인정보 처리방침 변경

본 방침은 법령·정책 변경 또는 서비스 변경에 따라 개정될 수 있습니다. 변경 시 앱 내 공지 및 이 페이지를 통해 시행일 7일 전부터 고지합니다.
