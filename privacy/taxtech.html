개인정보처리방침

TaxTech — 한국 세금 계산기

시행일: 2026년 1월 1일  |  최종 수정: 2026년 5월 7일

1. 수집하는 개인정보

TaxTech은 회원가입을 요구하지 않으며, 사용자를 직접 식별할 수 있는 개인정보(이름, 이메일, 전화번호, 주민등록번호 등)를 수집하지 않습니다.

2. 사용자가 입력하는 세금 계산 정보

본 앱은 세금 계산을 위해 사용자가 직접 입력한 정보(소득, 비용, 공제 항목 등)를 처리합니다. 해당 데이터는 다음과 같이 처리됩니다.

  • 입력된 모든 세금 계산 정보는 기기 내부에만 저장됩니다.
  • 외부 서버로 전송하거나 클라우드에 자동 동기화하지 않습니다.
  • 국세청·세무서 등 어떠한 외부 기관과도 데이터를 공유하지 않습니다.
  • 앱 삭제 시 모든 데이터가 함께 제거됩니다.

3. 기기 내 저장 데이터

  • 세금 계산 이력 — 소득세·부가세·환급액 계산 결과. 기기 내 로컬 스토리지(MMKV)에 저장.
  • 앱 설정 — 다크모드, 알림 설정, 약관 동의 여부. 기기 내 로컬 스토리지에 저장.
  • 구독 상태 — Pro 버전 활성화 여부. RevenueCat을 통해 익명 ID 기반으로 처리되며 개인 식별 정보는 포함되지 않습니다.

4. 알림 권한

세금 신고 마감일 알림 기능을 위해 알림 권한을 요청합니다. 모든 알림은 기기 내부에서 생성되며 외부 서버를 거치지 않습니다.

5. 광고 (Google AdMob)

본 앱은 Google AdMob 광고 SDK를 통해 광고를 표시할 수 있습니다. AdMob은 광고 식별자(IDFA/GAID), 광고 노출·클릭 통계, 대략적인 위치(국가 단위) 및 기기 정보를 처리할 수 있습니다.

iOS의 경우 앱 추적 투명성(ATT) 권한을 통해 사용자가 광고 추적 허용 여부를 선택할 수 있습니다. 거부 시에도 앱의 모든 기능을 정상 이용 가능합니다.

자세한 내용은 Google 광고 및 개인정보 정책을 참고해 주세요.

6. 인앱 결제 (RevenueCat)

Pro 구독 결제는 Apple App Store / Google Play Store를 통해 처리되며, RevenueCat 서비스를 통해 구독 상태가 관리됩니다. 결제 정보 자체는 개발사가 접근할 수 없으며, Apple/Google이 직접 처리합니다.

RevenueCat은 구독 상태 확인을 위해 익명화된 사용자 ID, 구매 영수증, 기기 OS 정보를 처리합니다. 자세한 내용은 RevenueCat 개인정보 정책을 참고해 주세요.

7. 분석 (Firebase Analytics)

앱 안정성 개선 및 기능 사용 현황 분석을 위해 Firebase Analytics를 사용합니다. 수집되는 정보는 익명화된 사용 통계(화면 방문, 기능 사용 빈도, 앱 충돌 정보)이며, 개인을 식별할 수 있는 정보는 포함되지 않습니다.

자세한 내용은 Firebase 개인정보 보호 페이지를 참고해 주세요.

8. 제3자 공유

위에 명시된 광고/결제/분석 SDK(Google AdMob, RevenueCat, Firebase) 외에는 어떠한 제3자에게도 사용자 데이터를 공유하지 않습니다. 위 SDK들은 각자의 개인정보처리방침에 따라 데이터를 처리합니다.

9. 세무 자문에 대한 면책

본 앱이 제공하는 세금 계산 결과는 참고용이며, 법적 효력이 있는 세무 자문이 아닙니다. 실제 세금 신고는 반드시 국세청 홈택스 또는 자격을 갖춘 세무사를 통해 진행하시기 바랍니다.

10. 어린이 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 개인정보를 수집하지 않습니다.

11. 보안

모든 세금 계산 데이터는 기기 내에만 저장되므로 별도의 서버 보안 위험이 없습니다. 기기 분실에 대비해 화면 잠금 등 기본 보안 조치를 권장합니다.

12. 방침 변경

본 방침이 변경될 경우 이 페이지에 최신 내용을 게시합니다. 중대한 변경 사항은 앱 내 공지를 통해 안내드립니다.
