개인정보처리방침

트래블환율 (Travel Currency Calculator)

시행일: 2025년 1월 1일  |  최종 수정: 2025년 5월 4일

1. 수집하는 개인정보

트래블환율은 어떠한 개인정보도 수집하지 않습니다.

  • 회원가입 없음 — 계정, 이메일, 이름, 전화번호 등 일체 수집하지 않습니다.
  • 광고 없음 — 광고 식별자(IDFA, GAID 등)를 수집하거나 제3자 광고 네트워크에 전달하지 않습니다.
  • 위치 정보 없음 — 기기 위치를 요청하거나 사용하지 않습니다.

2. 기기 내 저장 데이터

아래 데이터는 사용자의 기기 내부에만 저장되며, 외부 서버로 전송되지 않습니다.

  • 계산 히스토리 — 입력 금액, 환산 금액, 메모, 통화 코드, 시각. 기기 내 SQLite 데이터베이스에 저장.
  • 앱 설정 — 선택한 통화, 다크모드 설정, 햅틱 설정. 기기 내 로컬 스토리지에 저장.
  • 환율 캐시 — 마지막으로 받아온 환율 데이터. 오프라인 사용을 위해 기기에 임시 저장.

3. 환율 데이터 조회

앱은 최신 환율을 가져오기 위해 ExchangeRate-API(exchangerate-api.com)에 네트워크 요청을 보냅니다.

  • 전송 정보: 없음 (단순 GET 요청, 사용자 식별 정보 미포함)
  • 요청 빈도: 하루 1회 (이후에는 캐시 사용)
  • 인터넷이 없는 경우: 마지막으로 저장된 환율로 오프라인 동작

4. 제3자 공유

수집된 개인정보가 없으므로, 제3자에게 제공하는 데이터가 없습니다.

5. 어린이 개인정보 보호

트래블환율은 만 14세 미만 아동을 대상으로 개인정보를 수집하지 않으며, 해당 연령의 사용자에게 개인정보 제공을 요청하지 않습니다.

6. 보안

모든 데이터는 기기 내에만 저장되므로, 별도의 서버 보안 위험이 없습니다. 기기 보안은 운영체제(iOS/Android)의 샌드박스 정책을 따릅니다.

7. 방침 변경

본 방침이 변경될 경우 이 페이지에 최신 내용을 게시합니다. 중요한 변경이 있을 경우 앱 업데이트 릴리즈 노트를 통해 안내합니다.
